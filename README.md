# 🚗 Fresh Glory Motors – Car Rental Website

## 📌 Project Overview
Fresh Glory Motors is a fully responsive car rental website designed to provide users with a smooth and professional experience when browsing and booking vehicles. The platform showcases different car options, services, and helpful travel content.

This project was built as part of a web development assignment, focusing on modern UI/UX design, responsiveness, and clean structure.

---

## 🎯 Objectives
- Design a realistic multi-page website
- Ensure full responsiveness (desktop, tablet, mobile)
- Apply modern UI/UX principles
- Create a clean and user-friendly interface

---

## ⚙️ Challenges Faced
- Managing multiple pages with a single CSS file
- Ensuring consistent design across all sections
- Fixing layout issues caused by conflicting styles
- Implementing responsive design without breaking UI

---

## 💡 Solutions
- Organized CSS into reusable components
- Used Flexbox for flexible layouts
- Applied a structured responsive system (desktop-first)
- Cleaned and optimized duplicate styles
